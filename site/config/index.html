Configuración del Sistema

Esta sección describe las constantes, variables de entorno y parámetros técnicos que rigen el comportamiento del pipeline en todas sus etapas de ejecución.

Configuración General

Define las variables globales del proyecto, incluyendo las rutas base del sistema de archivos, credenciales de acceso a bases de datos y la configuración del broker de mensajería (Redis).

Configuración General del Proyecto. Gestiona las rutas del sistema, límites de archivos y políticas de reintentos.

CSV_DIR = os.path.join(RESOURCES_DIR, 'csv') module-attribute

Ruta al directorio de CSV

LARGE_FILE_THRESHOLD = 100 * 1024 * 1024 module-attribute

Umbral para considerar un archivo como 'grande' (por defecto 100MB).

LOG_DIR = os.path.join(RESOURCES_DIR, 'logs') module-attribute

Ruta al directorio de logs

MAX_RETRIES = 5 module-attribute

Número máximo de reintentos para operaciones fallidas en la pipeline.

PROJECT_DIR = os.path.abspath(os.path.join(os.path.dirname(__file__), '..', '..')) module-attribute

Ruta absoluta al directorio raíz del proyecto.

REPORTS_DIR = os.path.join(RESOURCES_DIR, 'reports') module-attribute

Ruta al directorio de reportes

RESOURCES_DIR = os.path.join(PROJECT_DIR, 'resources') module-attribute

Ruta al directorio de recursos

RETRY_DELAY = 60 module-attribute

Tiempo de espera (en segundos) entre reintentos.

TMP_DIR = os.path.join(RESOURCES_DIR, 'tmp') module-attribute

Ruta al directorio temporal

Parámetros por Fase de Procesamiento

La configuración está segmentada por fases para permitir un ajuste fino de los umbrales de procesamiento, extensiones de archivo permitidas y lógica específica de cada etapa.

Parámetros relacionados con el escaneo inicial del sistema de archivos y la profundidad del rastreo.
